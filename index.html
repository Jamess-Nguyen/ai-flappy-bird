This is a complete browser-based Flappy Bird game with an AI controller that uses a hybrid navigation strategy:

  • Top Half Gaps: Apex targeting - maintains altitude so jump apex threads through 3/4 of gap height
  • Bottom Half Gaps: Bottom pipe bouncing - treats bottom pipe like floor, bouncing along edge
  • Priority System: Floor safety ‚Üí Emergency collision ‚Üí Strategy selection
  • Predictive Physics: 5-frame lookahead for emergency collision avoidance
